# Demo 场景：Web 登录功能实现

## 场景描述

帮我实现一个简单的 Web 登录功能，包括以下要求：

1. **UI 设计**
   - 创建登录页面
   - 邮箱输入框（type="email"）
   - 密码输入框（type="password"）
   - 登录按钮
   - 错误提示显示区域

2. **表单验证**
   - 验证邮箱格式（使用正则表达式）
   - 验证密码强度（至少8个字符）
   - 检查字段是否为空

3. **错误提示**
   - 验证失败时显示具体错误信息
   - 错误信息清晰易懂
   - 错误提示及时显示

4. **测试验证**
   - 测试正确登录场景
   - 测试无效邮箱场景
   - 测试密码太短场景
   - 测试字段为空场景

## 预期成果

1. HTML 文件：login.html（登录页面结构）
2. JavaScript 文件：login.js（验证逻辑）
3. 所有功能通过 QA 测试验证

## 验收标准

- 页面 UI 设计清晰美观
- 表单验证逻辑正确
- 错误提示用户体验好
- 所有测试用例通过
